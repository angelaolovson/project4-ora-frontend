## User Stories
### MVP Goals
* As a user(customer), I want the ability to create an account and log in.
* As a user(customer), I want the ability to edit my account info.
* As a user(customer), I want to view all the products.
* As a user(customer), I want view each product's details.
* As a user(customer), I want the ability to add products in the shopping cart or delete it.
* As a user(customer), I want the ability to place an order and review the receipt.
* As a user(owner), I want the ability to create, edit, or delete products.
* As a user(owner), I want the ability to view all products when managing inventory.



### Stretch Goals
* Add address API + fake payment API to the check out page.
* Save products as wish list.
* Filter and sort function.
* Edit quantity in the shopping cart.
